# Banksight – Banking Analytics Dashboard

## Overview
Banksight is a banking analytics dashboard built using SQL, Python, Pandas, and Streamlit to analyze customer and account data and generate business insights.

## Tech Stack
- Python
- Pandas
- MySQL
- Streamlit

## Features
- Customer Analysis
- Account & Balance Insights
- Branch Performance Analysis
- Interactive Dashboard
